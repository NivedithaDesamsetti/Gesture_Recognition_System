# Gesture Recognition System

## Description
Gesture Recognition System is a web-based application developed using HTML, CSS, and JavaScript. The project helps deaf and dumb people communicate using hand signs. It detects hand gestures through a webcam and displays the meaning of the detected sign. The system also includes learning modules with sign images to help users learn sign language easily.

---

## Features
- Home page
- About page
- Learn sign language section
- Live gesture detection
- Webcam-based sign recognition
- Training module
- Settings page
- Real-time hand sign detection
- User-friendly interface
